Bem Vindo!

 

 

Projeto Web Fullstack completo com tecnologias como Spring Boot, Angular 10 e 11, Heroku, JPA, Git, Github, MySQL

O Site fornece um gerenciamento de livros, onde podemos listar, criar, editar e excluir livros e categorias.

 

Toda a aplicação é organizada em commits sobre cada parte do projeto.
Link

 

Backend esta hospedado na plataforma Heroku: Ao utilizar o sistema e não aparecer as categorias, aguarde 30 segundos para os itens serem carregados.
